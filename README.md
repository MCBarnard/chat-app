# Chat App
An online chat platform, checkout the [demo video](https://vimeo.com/695702990)



## Setup locally
* `git clone`
* `composer install`
* `php artisan key:generate`
* `php artisan migrate`
* `npm install`
* `npm run dev`
* `php artisan serve`
----------------------------
##### Optional to add Users
* `php artisan db:seed --class=CreateUsersSeeder`
----------------------------
